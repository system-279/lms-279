# 介護DX college ２７９ Classroom プライバシーポリシー

本プライバシーポリシー（以下「本ポリシー」）は、[事業者名を記入] （以下「当社」）が提供する「介護DX college ２７９ Classroom」（以下「本サービス」）における、利用者の個人情報の取扱いについて定めるものです。

## 第1条 基本方針

当社は、個人情報の保護に関する法律（個人情報保護法）その他関係法令を遵守し、本サービスにおいて取得する個人情報を適切に取り扱います。

## 第2条 取得する情報

本サービスの利用にあたり、以下の情報を取得します。

- メールアドレス、氏名（表示名）
- Google アカウント認証に基づくプロフィール情報（email_verified 等）
- サービス利用履歴（動画視聴時間、視聴位置、テスト回答・結果、出席記録）
- アクセスログ（IP アドレス、ブラウザ情報、アクセス日時）
- Cookie、ローカルストレージ情報

## 第3条 利用目的

取得した情報は以下の目的で利用します。

1. 本サービスの提供および運営
2. 受講進捗・テスト結果の管理および受講履歴の記録
3. 受講状況に基づく運営上の連絡およびサポート
4. 不正利用の検知・防止
5. サービス改善、統計分析
6. その他、利用者の同意を得た目的

## 第4条 第三者提供

当社は、以下の場合を除き、取得した情報を第三者に提供しません。

1. 利用者の同意がある場合
2. 法令に基づく場合
3. 人の生命、身体、財産の保護のために必要であり、本人の同意を得ることが困難な場合
4. 業務委託先への必要最小限の提供（例: クラウドサービス事業者）

## 第5条 外部サービスの利用

本サービスは以下の外部サービスを利用しており、これらのサービスを通じて一部の情報が取得・保管されます。

- Google Cloud Platform（Firestore、Cloud Storage、Cloud Run 等）
- Firebase Authentication（Google ログイン）
- Google Workspace（Drive、Docs 等との連携）

各サービスのプライバシーポリシーについては、各提供元にお問い合わせください。

## 第6条 保管期間

個人情報は利用目的の達成に必要な期間保管します。利用停止・退会の申出があった場合、合理的な期間内に削除します（法令で保管義務がある場合を除く）。

## 第7条 開示・訂正・削除等

利用者は、当社が保有する自己の個人情報について、開示・訂正・削除・利用停止を求めることができます。お問い合わせ窓口までご連絡ください。

## 第8条 安全管理措置

当社は、取得した情報の漏えい、滅失、毀損の防止その他安全管理のために必要かつ適切な措置を講じます。具体的には、アクセス権限の適切な付与、通信の暗号化（HTTPS）、認証基盤（Firebase Authentication + allowed_emails 方式）の運用等を行います。

## 第9条 Cookie の使用

本サービスは、認証状態の維持およびサービス品質向上のため Cookie を使用します。ブラウザ設定により Cookie を無効化することも可能ですが、一部機能が利用できなくなる場合があります。

## 第10条 未成年者の利用

未成年者が本サービスを利用する場合、法定代理人（保護者等）の同意を得た上で利用するものとします。

## 第11条 改定

本ポリシーは、法令の改正その他の理由により改定される場合があります。重要な変更がある場合は、本サービス上で通知します。
