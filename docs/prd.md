# PRD: LMS-279

## プロダクトビジョン

動画視聴の厳密な管理（倍速禁止・視聴追跡・完了判定）と入退室管理を実現する、ワンストップ型LMS。Google Classroomでは実現できない制御を自前で構築し、受講者の学習品質を保証する。

## なぜGoogle Classroomを使わないのか

Google Classroomは汎用的な教育プラットフォームだが、以下の要件を満たせない:

| 要件 | Google Classroom | LMS-279 |
|------|-----------------|---------|
| 倍速再生の禁止 | 不可（YouTube/Driveプレイヤー依存） | カスタムHTML5プレイヤーで完全制御 |
| 視聴区間の追跡 | 不可（再生開始/終了のみ） | heartbeat+イベントで秒単位追跡 |
| 視聴完了の厳密判定 | 不可 | coverageRatio算出、95%以上で完了 |
| 不審視聴パターン検出 | 不可 | サーバーサイドヒューリスティクス |
| 動画完了→テストのゲート | 不可（独立した機能） | video_analytics.isComplete連動 |
| 入退室の完全管理 | Classroom APIにログなし | セッション管理+heartbeat |
| マルチテナント | 組織ドメイン単位 | テナント単位の柔軟な分離 |

**結論**: 学習品質の保証に必要な「視聴行動の厳密な制御・追跡」がGoogle Classroomでは不可能なため、LMS機能を全て自前で構築する。OAuth審査コストの回避は副次的な理由。

## ターゲットユーザー

### テナント管理者（講師・教育担当者）
- 講座・レッスンの作成・管理
- 動画アップロード・テスト作成
- 受講者の進捗確認・不審視聴の把握
- 成績管理・CSVエクスポート

### 受講者
- 動画視聴（倍速不可、シーク検出）
- テスト受験（動画完了後にアンロック）
- 自分の進捗確認

### スーパー管理者
- 全テナントの管理・監視

## ユーザーストーリー

### 管理者
- 講座を作成し、レッスン（動画+テスト）を順序付きで構成できる
- 動画をアップロードし、必要視聴率（デフォルト95%）を設定できる
- テストを作成し、合格基準・最大受験回数・制限時間を設定できる
- 受講者の視聴進捗・テスト成績を一覧で確認できる
- 倍速再生やタブ離脱など不審な視聴パターンを把握できる

### 受講者
- 公開された講座のレッスンを順序通りに受講できる
- 動画を視聴し、完了後にテストを受験できる
- 自分の進捗（完了レッスン数、テスト成績）を確認できる

## 機能要件（優先度順）

### P0: Must Have（Phase 1-2）
- Firebase認証 + マルチテナント
- 講座・レッスンCRUD
- カスタム動画プレイヤー（倍速禁止、イベント追跡）
- GCS動画ホスティング + 署名付きURL
- サーバーサイド視聴分析（coverageRatio、完了判定）

### P1: Should Have（Phase 3-4）
- テスト作成・受験・自動採点
- 動画完了→テストゲート
- 進捗トラッキング（レッスン・コース単位）
- レッスンアンロック制御

### P2: Nice to Have（Phase 5）
- 管理ダッシュボード（分析・統計）
- 不審視聴パターン検出・表示
- CSVエクスポート
- デモモード + シードデータ

## 非機能要件

| 項目 | 基準 |
|------|------|
| レスポンスタイム | API P95 < 500ms |
| 動画配信 | 署名付きURLによるGCS直接配信 |
| スケーラビリティ | Cloud Runオートスケール |
| セキュリティ | テナントデータ分離、署名付きURL（2h有効）、サーバーサイド採点 |
| 可用性 | Cloud Runマネージド（SLA 99.95%） |

## スコープ外

- Google Classroom API連携（倍速禁止・視聴追跡が不可能なため不採用）
- Google Forms連携（テストは自前実装）
- リアルタイムコラボレーション（掲示板、チャット）
- 動画のライブ配信
- モバイルアプリ（Webレスポンシブで対応）
- 多言語対応（日本語のみ）

## 成功指標

| KPI | 目標 |
|-----|------|
| 動画完了率 | 受講者の80%以上が必要視聴率を達成 |
| 倍速違反検出率 | 試行の99%以上を即時リセット |
| テスト合格率 | 初回受験で60%以上が合格 |
| 管理者操作時間 | 講座+レッスン+テスト作成が30分以内 |
