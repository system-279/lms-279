# 要件定義

## 概要

動画視聴管理・テスト機能を統合したLMS（Learning Management System）。マルチテナント対応。

## ユーザーロール

| ロール | 説明 |
|--------|------|
| スーパー管理者 | 全テナント管理 |
| テナント管理者 | 自テナントの講座・受講者管理 |
| 受講者 | 動画視聴・テスト受験 |

## 機能要件

### 講座管理（管理者）
- 講座のCRUD（作成/読取/更新/削除）
- 講座ステータス管理（draft → published → archived）
- レッスンの順序管理

### 動画管理
- GCSへの動画アップロード（署名付きURL）
- 署名付きURLによる動画配信（2時間有効）
- カスタムHTML5動画プレイヤー
  - 倍速禁止（ratechange即時リセット）
  - 再生イベント追跡（play/pause/seek/ended/heartbeat）
  - タブ離脱検知（visibilitychange API）
- サーバーサイド視聴分析
  - 視聴区間算出・マージ
  - coverageRatio計算
  - 完了判定（coverageRatio >= requiredWatchRatio）
  - 不審パターン検出
- Google Driveからの動画インポート（GCSへコピー）

### 出席管理
- 入室打刻: 動画再生開始時に記録
- 退室打刻: テスト送信時に記録
- セッション制限時間: 入室から2時間（カウントダウン表示）
  - 残り30分: 黄色警告
  - 残り10分: 赤色警告（パルスアニメーション）
  - 超過時: 強制退室 + テスト送信拒否
- 一時停止制限: 15分以上の一時停止で強制退室
- 受講ルールの常時表示（各レッスンページ）
- 管理者向け出席一覧 + CSVエクスポート

### テスト
- テスト作成・編集（上限50問、single/multi選択）
- 動画完了がテストアクセスをゲート
- サーバーサイド自動採点（正解非公開）
- 制限時間、ランダム化対応
- 最大受験回数制限
- Google DocsからのAIテスト自動生成（Vertex AI / Gemini）

### 進捗トラッキング
- レッスン単位の進捗（動画完了 + テスト合格）
- コース単位の進捗（完了レッスン数/全レッスン数）
- レッスンアンロック制御

### 管理ダッシュボード
- コース別/受講者別進捗一覧
- 動画視聴統計
- テスト統計
- 不審視聴パターン表示
- 出席管理（入退室記録・強制退室履歴）
- CSVエクスポート（進捗・出席）

### 認証・アクセス制御
- Firebase Authentication + Googleソーシャルログイン
- アクセス許可リスト（allowed_emails）
- マルチテナント（URLパスベース）
- セルフサービステナント登録
- スーパー管理者

## 非機能要件
- レスポンスタイム: API < 500ms（P95）
- 動画配信: 署名付きURLで直接GCS配信
- スケーラビリティ: Cloud Runオートスケール
- セキュリティ: データテナント分離、署名付きURL、サーバーサイド採点
