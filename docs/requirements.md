# 要件定義

## 概要

動画視聴管理・クイズ機能を統合したLMS（Learning Management System）。マルチテナント対応。

## ユーザーロール

| ロール | 説明 |
|--------|------|
| スーパー管理者 | 全テナント管理 |
| テナント管理者 | 自テナントの講座・受講者管理 |
| 受講者 | 動画視聴・クイズ受験 |

## 機能要件

### 講座管理（管理者）
- 講座のCRUD（作成/読取/更新/削除）
- 講座ステータス管理（draft → published → archived）
- レッスンの順序管理

### 動画管理
- GCSへの動画アップロード（署名付きURL）
- 署名付きURLによる動画配信（2時間有効）
- カスタムHTML5動画プレイヤー
  - 倍速禁止（ratechange即時リセット）
  - 再生イベント追跡（play/pause/seek/ended/heartbeat）
  - タブ離脱検知（visibilitychange API）
- サーバーサイド視聴分析
  - 視聴区間算出・マージ
  - coverageRatio計算
  - 完了判定（coverageRatio >= requiredWatchRatio）
  - 不審パターン検出

### クイズ
- クイズ作成・編集（上限50問、single/multi選択）
- 動画完了がクイズアクセスをゲート
- サーバーサイド自動採点（正解非公開）
- 制限時間、ランダム化対応
- 最大受験回数制限

### 進捗トラッキング
- レッスン単位の進捗（動画完了 + クイズ合格）
- コース単位の進捗（完了レッスン数/全レッスン数）
- レッスンアンロック制御

### 管理ダッシュボード
- コース別/受講者別進捗一覧
- 動画視聴統計
- クイズ統計
- 不審視聴パターン表示
- CSVエクスポート

### 認証・アクセス制御
- Firebase Authentication + Googleソーシャルログイン
- アクセス許可リスト（allowed_emails）
- マルチテナント（URLパスベース）
- セルフサービステナント登録
- スーパー管理者

## 非機能要件
- レスポンスタイム: API < 500ms（P95）
- 動画配信: 署名付きURLで直接GCS配信
- スケーラビリティ: Cloud Runオートスケール
- セキュリティ: データテナント分離、署名付きURL、サーバーサイド採点
