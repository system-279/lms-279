# ADR-013: GCS動画ホスティング + 署名付きURL

## ステータス
承認済み

## コンテキスト
動画ファイルのストレージと配信方式の選定

## 決定
Google Cloud Storageに動画を保存し、署名付きURL（2時間有効）で配信

## 根拠
GCSは大容量ファイルの保存に最適。署名付きURLで認証済みユーザーのみ動画にアクセス可能。CDN統合も将来的に容易

## 影響
lms-279-videosバケット（動画保存）、lms-279-uploads（一時アップロード、24h TTL）。管理者はアップロードURL取得→直接GCSにアップロード→メタデータをAPI登録
